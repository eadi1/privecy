Privacy Policy – As Sunnah Guardian App Last updated: 07-01-2026 As Sunnah Guardian App (“the App”) is developed to allow guardians/parents to securely access student-related information of As Sunnah educational institutions. This Privacy Policy explains what data we collect, how we process it, and how we protect it. 1. User Accounts & Login Process To use the App, guardians must log in using authorized credentials provided or verified by the institution. The login process may include: Student ID Guardian phone number or username Password set by the guardian One-Time Password (OTP) verification (if enabled) Passwords are encrypted and never stored or transmitted in plain text. 2. Information We Collect We may collect the following information only as required for app functionality: a) Account Information Student ID Guardian phone number or email Login credentials (encrypted) b) Student Information (View-only) Student name, class, roll Attendance records Exam results Fee/payment status Notices and announcements c) Technical Information Device type and app version App usage logs (for error fixing and performance improvement) 3. How We Use Collected Information Collected data is used strictly to: Authenticate guardians securely Display student academic and institutional data Send important notifications and notices Maintain system security and prevent unauthorized access Improve app reliability and performance We do not use data for advertising or marketing. 4. OTP & SMS Usage (If Applicable) If OTP verification is enabled: Phone number is used only for authentication OTP messages are sent for login security Phone number is not shared with third parties OTP data is not stored after verification 5. Permissions Used The App may request the following permissions: Internet – for secure communication with servers SMS (optional) – for OTP auto-verification Storage (optional) – to download PDFs such as results or reports Permissions are requested only when required. 7. Data Security We use appropriate security measures including: Encrypted communication (HTTPS) Secure authentication mechanisms Restricted access to backend systems Only authorized personnel can access the data. 8. Children’s Privacy This App is intended for guardians/parents, not children. Student data is accessed only by verified guardians through secure login. 9. Data Retention & Deletion User data is stored only while the account is active Data may be deleted upon institutional request or account deactivation Backup retention follows institutional policy 10. Changes to This Privacy Policy We may update this Privacy Policy as needed. Any changes will be published on this page with an updated date. 11.
